# 🎮 Jogo Par ou Ímpar


## 📖 Sobre o Projeto

Este é o meu primeiro jogo desenvolvido na Unity utilizando **C#**! 

O clássico **Par ou Ímpar** ganhou vida com uma interface gráfica (UI) criada do zero. 
Foi meu primeiro passo no mundo do desenvolvimento de jogos com Unity e C#, onde pude colocar em prática conceitos fundamentais como:

- ✅ Criação de interfaces interativas (UI)
- ✅ Lógica de programação com C#
- ✅ Gerenciamento de eventos e cliques
- ✅ Manipulação de textos e botões

---

## 🎯 Como Jogar

1. Monte a cena localizada em assets -> scenes -> desafio 8
2. Digite seu nome, sua escolha e o que você vai jogar, tudo em minúsculo
3. A CPU sempre vai escolher a opção oposta a sua, se você for par ela automaticamente escolhe ímpar e vice-versa.
4. Clique no botão 'jogar' e boa sorte 🎲
